# 프로젝트 개요

## 목적

이 기준 모델은 CEMS 전력 사용량 예측의 첫 번째 모델링 단계다. 목표는 처음부터 가장 높은 정확도를 만드는 것이 아니다. 전력 사용량 데이터만으로 어느 정도의 예측 성능을 확보할 수 있는지 확인하는 것이다.

## 모델링 로드맵

### 1단계: 전력 데이터 기반 기준 모델

과거 전력 데이터만 사용한다.

- 학교 전체 일별 사용 패턴
- 건물별 월별 사용량 프로필
- 검증용 최근 건물별 일별 사용량

이 단계의 결과는 이후 모든 모델의 비교 기준이 된다.

### 2단계: 외부 변수 추가

전력 데이터만 사용한 기준 모델의 성능을 확인한 뒤 외부 변수를 추가한다.

- 기상: 기온, 습도, 강수량
- 달력: 요일, 주말 여부, 공휴일
- 학사일정: 학기 중 여부, 방학, 시험 기간

### 3단계: 머신러닝 모델 학습

기준 모델과 외부 변수가 준비된 뒤 LightGBM 같은 머신러닝 모델을 적용한다. 이때 단순히 오차 수치만 보는 것이 아니라, 전력 데이터 기반 기준 모델보다 얼마나 개선되었는지를 비교한다.

## 현재 범위

포함한다.

- `primary_reliable` 건물 17개
- 건물별 일일 전력 사용량 예측
- `2026-04-01`부터 `2026-06-23`까지의 검증
- 해석 가능한 단순 기준 모델과 성능 지표

1차 기준 모델에서는 제외한다.

- 기상 변수
- 학사일정 변수
- LightGBM 또는 기타 머신러닝 모델
- 실시간 웹 서비스 연동
