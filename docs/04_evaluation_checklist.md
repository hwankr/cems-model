# 평가 체크리스트

모델 결과를 보고서나 프로젝트에 사용하기 전에 이 체크리스트를 확인한다.

## 데이터 확인

- [ ] 메인 입력 파일이 `10_baseline_ready_panel_primary_reliable.csv`다.
- [ ] 건물 수가 17개다.
- [ ] 날짜 범위가 `2026-04-01`부터 `2026-06-30`까지다.
- [ ] 검증 행은 `is_validation_target_clean == True`를 사용한다.
- [ ] 검증 구간에서 `2026-06-24` 부분 일자를 제외한다.
- [ ] 검증 구간에서 `2026-06-25`부터 `2026-06-30`까지의 placeholder 날짜를 제외한다.

## 예측값 확인

- [ ] 균등 일할 기준 모델 예측값이 생성된다.
- [ ] 학교 전체 일별 패턴 기준 모델 예측값이 생성된다.
- [ ] 예측값 컬럼 전체가 결측인 경우가 없다.
- [ ] 예측값은 음수가 아니다.
- [ ] placeholder 날짜는 평가하지 않지만 예측값은 생성할 수 있다.

## 성능 지표 확인

- [ ] MAE를 계산한다.
- [ ] RMSE를 계산한다.
- [ ] MAPE를 계산하되 실제값이 0인 행은 제외한다.
- [ ] Bias를 계산한다.
- [ ] 건물별 성능표를 저장한다.
- [ ] 월별 성능표를 저장한다.
- [ ] 오차가 큰 행 목록을 저장한다.

## 해석 확인

- [ ] 전체적으로 더 나은 기준 모델 방식을 판단한다.
- [ ] 지속적으로 오차가 큰 건물을 확인한다.
- [ ] 오차가 큰 월이나 날짜 구간을 확인한다.
- [ ] 전력 데이터만으로 다음 단계에 충분한지 판단을 기록한다.
- [ ] 다음 단계에서 추가할 외부 변수를 기록한다.
