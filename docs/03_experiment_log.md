# 실험 로그

이 파일은 모델 실험의 실행 기록 인덱스로 사용한다.

상세 실행 기록이 필요하면 [templates/experiment_entry.md](templates/experiment_entry.md)를 복사해 `docs/runs/YYYY-MM-DD-run-name.md` 형식으로 저장한다.
