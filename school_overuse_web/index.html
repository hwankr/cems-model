개요

검증 기간 데이터 기반 과다사용 요약

이 대시보드는 학교 건물 전체의 시간별 전력 사용량을 day-ahead(하루 전) 예측 모델과 비교하여, 어느 시간이 정상 범위(P10~P90 정상밴드)를 초과하는 과다사용인지 알려줍니다. 모델은 과거 사용 패턴, KMA 날씨 예보, 학사일정을 입력으로 받아 LightGBM 분위수 회귀로 P10·P50·P90을 예측하며, 최근 28일(5월) 데이터를 활용한 CQR(Conformal Quantile Regression) 보정을 통해 정상밴드가 실제값을 80% 적중하도록 조정합니다. 실제가 P90을 넘으면 과다사용이며, 초과량(kWh)과 그 시간의 SHAP 예측 근거를 함께 제공합니다.

용어 설명

대시보드에서 사용하는 핵심 개념 안내

분위수(Quantile)

데이터를 크기순으로 정렬했을 때 특정 비율 위치의 값. P90 = 상위 10% 경계값입니다.

P10 / P50 / P90

이 시간에 '정상이라면' 사용량이 P10 이상일 확률 90%, P50(중앙값)을 넘을 확률 50%, P90을 넘을 확률 10%입니다. P90을 넘으면 과다사용으로 판정합니다.

정상 밴드 / 과다사용

[P10, P90]이 정상 범위(정상밴드)입니다. 실제 > P90이면 과다사용이며, 초과량 = 실제 − P90(kWh)입니다.

day-ahead(하루 전 기준)

당일 직전 관측을 사용하지 않고 날짜·요일·시간·예보 날씨·학사일정·과거 부하만으로 다음 날 전체를 예측합니다. 실제 운영 환경에서 사용 가능한 공정한 예측 방식입니다.

SHAP

트리 모델 예측을 각 피처가 얼마나 P50 기대값을 끌어올리거나 낮췄는지 kWh 단위로 분해한 값입니다. 빨강 = 기대를 높인 요인, 파랑 = 낮춘 요인.

WAPE

가중 절대 백분율 오차 = Σ|예측−실제| / Σ실제. 낮을수록 P50 예측이 정확합니다. 이 모델은 7.6%로 단순 naive 대비 약 40~50% 오차 감소를 달성합니다.

Pinball loss

분위수 예측 품질 지표. 실제값이 분위수 아래/위에 있을 때 비대칭 패널티를 부여합니다. 낮을수록 분위수가 잘 맞습니다.

Coverage(적중률)

실제값이 [P10, P90] 안에 든 비율. CQR(Conformal Quantile Regression) 보정으로 ≈80%를 목표합니다.

CDD/HDD

냉방·난방 도일(degree-day). 기준온도(18°C/20°C) 대비 더운/추운 정도의 누적 지표. 여름 냉방 수요 예측에 중요한 피처입니다.

CQR(Conformal Quantile Regression)

분위수 예측을 보정하는 통계적 방법. 검증(보정) 데이터에서 오차 분포를 추정해 밴드 폭을 조정함으로써 실제 적중률이 목표(80%)에 맞도록 합니다.
